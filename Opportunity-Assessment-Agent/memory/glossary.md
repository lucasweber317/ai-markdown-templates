# Glossary

> This file defines product terminology to ensure consistent language across assessments and Epics.

---

## Terms

- **Epic**: A structured initiative that delivers measurable value.
- **Opportunity**: A problem or strategic initiative under evaluation.
- **Primary Persona**: The main user impacted by a change.
