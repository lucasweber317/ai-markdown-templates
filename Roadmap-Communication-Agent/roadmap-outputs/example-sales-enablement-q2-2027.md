## Executive Summary

Q2 brings enterprise security certifications, major performance improvements, and modernized admin experience. These capabilities directly address the top 3 objections we hear in enterprise deals: security compliance gaps, performance at scale, and admin complexity. Sales can begin positioning these capabilities in active deals with Q2 availability.

## FAQ

**Q: When exactly in Q2 will SOC 2 certification be complete?**
A: Audit is in progress. Expected completion mid-Q2 (May timeframe). Report will be available for customer security reviews upon completion.

**Q: Can I tell customers about SOC 2 certification in active deals?**
A: Yes, but specify "completing audit in Q2 2027" if asked for timeline. Don't promise specific dates. Report will be shareable once audit finalizes.

**Q: How much faster will performance be after Q2 optimization?**
A: 50% improvement in API response times; 10x increase in data volume capacity. Exact numbers depend on customer usage patterns—use ranges, not guarantees.

**Q: Do the new integrations require custom development?**
A: No. Pre-built connectors for Salesforce, HubSpot, Zendesk. Plug-and-play setup. Customer just provides API credentials.

**Q: Will there be additional cost for the new integrations?**
A: No additional cost. Included in existing Enterprise plan. Use this as competitive advantage vs. solutions charging per-connector fees.

**Q: What about the admin UI redesign—will customers need retraining?**
A: Minimal retraining needed. New UI is more intuitive—customers report faster configuration in beta testing. We'll provide migration guide and training materials.
