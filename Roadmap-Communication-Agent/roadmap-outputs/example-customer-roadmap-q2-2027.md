## Q2 2027: What's Coming

| Capability | What It Does for You |
|-----------|---------------------|
| **Enterprise Security Certification** | We're completing SOC 2 Type II certification so you can confidently pass your organization's security reviews; streamlines your compliance audits with third-party validated controls |
| **Role-Based Access Controls** | You can restrict features and data access by user role—your security team controls who sees sensitive information; reduces risk from overprivileged users |
| **Faster Performance** | 50% faster response times and support for 10x more data—platform grows with your organization without slowdowns; improves experience for your end users |
| **Redesigned Admin Experience** | New configuration interface with bulk operations and intuitive workflows; saves hours on setup and ongoing management; faster onboarding for new admins |
| **Pre-Built Integrations** | Ready-to-use connectors for Salesforce, HubSpot, and Zendesk; no custom development needed; unified workflows across your tools with plug-and-play setup |
