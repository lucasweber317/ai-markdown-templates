New features and enhancements

**Advanced Filtering System**

You can now apply up to 10 filters simultaneously across data tables to quickly find specific records. The new filtering system supports multiple operators (equals, contains, greater than, less than) and saves your filter configurations across sessions.

This is especially useful if you manage large datasets with 10,000+ records, eliminating the need to export data to Excel for complex filtering. Filter configurations can also be exported and imported for reuse across your team.

Available in all plan tiers. Requires Chrome 90+, Firefox 88+, or Safari 14+.

**Bulk Action Operations**

Select up to 500 items and perform bulk operations including status updates, tag management, exports, and archiving. This eliminates repetitive manual actions that previously required clicking through hundreds of items individually.

Bulk actions are available to Professional and Enterprise plan customers. Bulk delete operations require Admin role permissions.

**Enhanced API Rate Limits for Enterprise**

Enterprise customers now have access to 5,000 API requests per hour, increased from 1,000 requests per hour. This supports high-volume integrations and continuous sync operations without throttling.

Available to Enterprise plan customers using API key version 2.0 or higher. Standard plan limits remain at 1,000 requests per hour.
