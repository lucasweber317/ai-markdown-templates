## Action Verbs and Phrasing

Example entries:

- Use "Available" not "Rolled out" or "Shipped"
- Use "Enhanced" for improvements, "Added" for new features, "Fixed" for bug repairs
- Use "Customers can now..." or "You can now..." for action-oriented descriptions
- Avoid passive voice: "The feature was improved" → "We improved the feature"
